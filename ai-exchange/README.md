# OWASP AI Exchange

- OWASP が公開している [AI Exchange](https://owaspai.org/) を勝手に（＝非公式）日本語にしてみたものです。参考までにどうぞ・・・
- 文書の章立ては以下のとおりです（英語原文に従う）。日本語化は、私が個人的に興味のある分野優先です。
  - 00. AI Security Overview（AI セキュリティの概観）
  - 01. General Controls（一般的な管理策）
  - 02. Threats Through Use（利用時の脅威）
  - 03. Develop-time Threats（開発時の脅威）
  - 04. Runtime Application Security Threats（実行時のアプリケーション セキュリティ脅威）
  - 05. AI Security Testing（AI セキュリティ テスト）
  - 06. AI Privacy（AI プライバシー）
  - 07. AI Security References（AI セキュリティの参考情報）
